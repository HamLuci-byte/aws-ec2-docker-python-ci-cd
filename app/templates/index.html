HOW SUMMY WORKS ?

01.

Text Pre-Processing

Transformation of text into a more digestible form so that machine learning algorithms can perform better.

  • Lemmatization
  • Stemming
  • POS tagging
02.

Sentence Score

Sentence is assigned a value to rank it. The value is assigned based on several attributes.

  • Text-Rank
  • Word Frequency
  • Tf-Idf Score
03.

Generation

Based on the sentence score, summary is generated. Thresholds are applied and summary is generated.

  • Discarding
  • Merging
  • Results
